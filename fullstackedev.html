Course Overview

Module 1:Introduction to Full Stack Developement

What is Full Stack Developement

  • A Full Stack Developer is someone who can work on both ends of a web application.
  • designing the user interface to handling the server, database, and APIs.

Basic Tools

  • Code Editors / IDEs
  • Web Browsers
  • Frontend Development Tools
  • Backend Development Tools
  • Database Tools
  • Deployment / Hosting Platforms

Module 2: Main Components of Full Stack

  • Frontend → sends a request to server
  • Backend → processes the request
  • Database → stores/fetches data
  • Backend → sends response back
  • Frontend → shows data to the user

Module 3: What is Backend Development?

Backend Development refers to the server-side of a web application. It is responsible for:

  • Business logic
  • Database communication
  • User authentication
  • Server handling
  • API creation

Basic of database

  • Save user information (name, email, password)
  • Store product details, orders, messages, etc.
  • Retrieve data quickly when users request it

Module 4: Introduction to GitHub

What is GitHub?

  • Store their code
  • Track changes in their projects
  • Collaborate with other developers
  • Host open-source or private projects
