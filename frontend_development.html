Course Overview

Module 1: Understand Web Fundamentals

What is Web Fundamentals?

  • Explain how websites work (Client vs Server)
  • Understand the role of frontend in web development
  • Know the difference between frontend and backend

Build Webpage Structure with HTML

  • Create semantic and well-structured HTML pages
  • Use tags like headings, paragraphs, forms, tables, images, and links
  • Apply best practices for accessibility and SEO-friendly structure

Module 2: Make Websites Interactive with JavaScript

Introduction with JavaScript

  • Use variables, functions, conditions, and loops
  • Handle events like clicks, form submissions, etc.
  • Manipulate webpage content using DOM (Document Object Model)

Use Frontend Frameworks & Libraries

  • Build responsive UI using Bootstrap
  • Understand the basics of using JavaScript libraries (e.g., jQuery, optionally React for advanced learners)
