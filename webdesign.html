Welcome!

This course offers an engaging and interactive approach for kids to learn the fundamentals of web design. Through hands-on practice with HTML, CSS, and JavaScript, students will unleash their creativity while developing essential skills to create their own websites and understand the principles of effective web design in a fun environment.

Prerequisites

No prior experience in web design is required. This course is designed for kids, offering an engaging introduction to HTML, CSS, and JavaScript. Participants will develop essential skills through hands-on practice, fostering creativity and confidence in web design

Learning Objectives

This course offers a fun and interactive environment for kids to learn fundamental web design concepts. It fosters creativity and hands-on practice, empowering students to create their own web pages using HTML, CSS, and JavaScript while building essential coding skills.

Course Overview

Module 1: Introduction to Web Design

What is a Website?

  • How Websites Work (Browser, Server, Internet)
  • Introduction to Web Pages

Basic Tools

  • Text Editors (e.g., Notepad, VSCode)
  • Browser (e.g., Chrome, Firefox)

Module 2: HTML Basics

Introduction to HTML

  • What is HTML? HTML Structure
  • Creating a Basic Web Page (Head, Body)

Common HTML Elements

  • Headings, Paragraphs, Links, Images
  • Lists (Ordered, Unordered)
  • Tables and Forms (Simple)

Module 3: Styling with CSS

Introduction to CSS

  • What is CSS? How to Add CSS to HTML (Inline, Internal, External)

Basic CSS Styling

  • Changing Text Colors, Fonts, and Sizes
  • Background Colors and Images
  • Padding, Margins, and Borders

Creating Simple Layouts

  • Aligning Content with CSS
  • Using Flexbox for Basic Layout

Module 4: Introduction to JavaScript

What is JavaScript

  • Adding JavaScript to Web Pages
  • Using the Console for Simple Outputs

Basic JavaScript Concepts

  • Variables, Data Types (Numbers, Strings)
  • Simple Math Operations
  • Using Alerts and Prompts for Interaction
